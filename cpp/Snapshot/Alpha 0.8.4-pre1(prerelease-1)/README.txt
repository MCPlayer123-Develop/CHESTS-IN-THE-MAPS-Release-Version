打开CITM_a0.8.4_pre1.exe(应用程序)，便可正常游玩
Open CITM_a0.8.4_pre1.exe, then you can enjoy the game.
或者先打开项目，再运行Code\main.cpp，便可正常游玩
Please open the project, then run "Code\main.cpp", then you can enjoy the game.
注意：
Warning:
如果小熊猫C++运行的中文乱码，请把main.cpp中的SetConsoleOutputCP(65001);改为SetConsoleOutputCP(936);。
If there are garbled Chinese characters when running the panda C++ program, change SetConsoleOutputCP(65001); to SetConsoleOutputCP(936); in main.cpp;.
